# saranen-lottokone-harjoitus

Lottokone

Install Node.js

If you want 7 random numbers, run

`node . random`

If you want to choose your own numbers, run

`node . $1 $2 $3 $4 $5 $6 $7`

where $# is your number. Number can be whole number between 1 and 40, inclusive.

Example:

`node . 23 1 36 31 29 12 15`
